# Veri Seti

## Varsayilan Davranis
Script, bu klasorde `.csv` dosyasi bulamazsa **gomulu ornek veri** (40 Turkce cumle) ile calisir.

## Kaggle'dan Veri Indirme (Opsiyonel)

Daha buyuk bir veri seti ile calistirmak icin:

1. [Turkish Sentiment Analysis Dataset](https://www.kaggle.com/datasets/winvoker/turkish-sentiment-analysis-dataset) adresinden CSV'yi indirin.
2. Dosyayi bu klasore (`data/`) koyun.
3. CSV'nin `text` ve `label` (veya `sentiment`) sutunlari icerdiginden emin olun.

Script otomatik olarak CSV'yi bulup yukleyecektir.
